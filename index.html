Minley

A JavaScript-powered Discord bot bringing essential Minecraft monitoring and utility features straight to your server.

100% Free Hosting Not 24/7 Prefix: mc!
Current Prefix: mc!
[...] = Optional <...> = Required
mc!status [ip]
Draws a custom status card for a server.
mc!players [ip]
Lists online players on a server.
mc!plugins [ip]
Detects server software and plugins.
mc!skin <user>
Fetches a 3D full-body skin render.
mc!uuid <user>
Gets the official Mojang account UUID.
mc!item <n> | <+>
Renders an authentic item tooltip box.
mc!stagemod
Toggles joining Stage Channels. Owner Only
